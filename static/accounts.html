Google аккаунты

Каждый аккаунт = один OAuth-токен под коротким именем. Агент может работать с любым из них одновременно.

Добавить аккаунт
Введите короткое имя (например work, elena) — откроется браузер для входа в Google.
Если ловите "Access blocked: claude-mcp has not completed verification":
Google нужно сначала разрешить этому email авторизоваться. Откройте консоль проекта, найдите Test users → Add users → впишите тот email, под которым будете логиниться → Save. Затем попробуйте Добавить аккаунт снова.
